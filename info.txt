El sitio web debe incluir:

# Estructura

* <header>
* <nav> (puede estar dentro de header)
* <main>
* <footer>

---

# Contenido

* Título principal (`h1`)
* Subtítulos (`h2`)
* Párrafos
* Una lista (`ul` o `ol`)

---

# Enlaces: Puede crear más de una página (no muchas) dentro del mismo proyecto y enlazarlas entre ellas

* Un enlace externo (por ejemplo, a otra página web) 
* Un enlace interno (a otra página o sección) - recuerde el uso de id="" dentro de la etiqueta correspondiente

---

# Multimedia: agregar carpeta al proyecto img o assets para guardar los recursos

* Una imagen
* Un video utilizando <iframe> o audio con <audio controls>.

---

# Tabla

* Debe incluir encabezados (`th`)
* Al menos 2 filas
* Uso de `colspan` o `rowspan`

---

# Formulario

* Campo de nombre
* Campo de email
* Campo de mensaje
* Botón de envío

---

# Elementos adicionales (al menos uno)

* Menú de navegación
* <aside>
* <article>
* Favicon

---

# README.md

El repositorio debe incluir un archivo `README.md` con:

* Nombre del proyecto
* Breve descripción
* Contenido incluido
* Nombre del autor
