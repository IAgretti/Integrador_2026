Etiquetas de formato

El siguiente texto tiene ejemplos de uso de etiquetas de formato, muy útiles para texto en línea, cómo "cursiva o itálica", negrita o la otra similar negrita. También podemos utilizar énfasis que depende el navegador pero es similar a "i".

En este fragmento se muestran algunas etiquetas de formato más, por ejemplo: código para fragmentos de código y tecla para atajos de teclado. Texto tachado, subíndice y superíndice. Son buenas etiquetas para marcar el texto en partes específicas.
